# Video compression specs:

## Desktop (16:9 landscape)

Resolution: 1920 x 1080 (or 1620 x 1080)

Handbrake settings:
- Summary
  - Format: MP4
  - Web optimized: checked
- Dimensions
  - Resolution limit: custom: 1920 x 1080
- Video
  - Video Encoder: H.264 (x264)
  - Framerate: 30fps
  - Quality: Constant Quality: RF32
- Audio: none
